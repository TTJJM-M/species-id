# 物种识别 (Species ID)

一个个人用的移动端物种识别工具。用户拍摄照片后，自动识别图片中的主体内容并给出名称、简介、类别和置信度。

## 领域 (Domain)

模型根据图片内容自动判断的类别名（如植物、动物、菜品、建筑、车辆等），不再限定为固定三类。

_Avoid_: 类别、分类、category

## 物种 (Species)

识别结果中最核心的信息，以该物种的中文常用名表示（如"银杏"、"柯基"、"宫保鸡丁"），不作为学名。

_Avoid_: 种类、品种、名称

## 置信度 (Confidence)

模型对自己识别结果给出的确定性，0 到 100 的整数。高于等于 55 才采纳结果，低于 55 判定为认不准。

_Avoid_: 准确率、把握度、概率

## 认不准 (Unrecognized)

置信度低于 55 时的结果状态。不为领域/物种做任何猜测，如实告知用户无法识别并建议更换角度。这不是错误——模型正常运行，只是不够确定，与网络故障是不同性质的状态。

_Avoid_: 识别失败、未知、错误

## 图鉴 (Field Guide)

用户所有识别成功记录的集合，以列表形式呈现（最近的在最上面），支持按领域筛、按物种名搜索。每一条包含缩略图、物种名、拍摄时间。点进入可看原图缩放 + 完整信息。用户可删除不需要的条目。

_Avoid_: 历史记录、收藏夹、相册

## API Key

云端模型服务的访问凭证，**只存在于后端，不在 app 中**。app 无权直接获取或持有此密钥。

_Avoid_: token、密钥、secret
